## 📋 Описание проекта

Современный, стильный и солидный одностраничный лендинг для группы компаний **САМАДИ** - ведущего дистрибьютора мировых и отечественных брендов в Казахстане.

### 🎯 Основная цель
Презентация компании САМАДИ как надежного партнера с 30-летним опытом работы на рынке Казахстана, демонстрация ключевых показателей, компетенций и географии присутствия.

## ✨ Основные возможности

### 🏆 Реализованные функции

#### 1. **Hero-секция с впечатляющим визуалом**
- Полноэкранный баннер с фоновым изображением современного склада
- Анимированные ключевые показатели (30 лет, 500 сотрудников, 150К заказов)
- Градиентный текстовый эффект для названия компании
- Индикатор прокрутки с анимацией

#### 2. **О компании**
- 3 карточки с ключевыми преимуществами:
  - Лидер рынка
  - Долгосрочное партнерство (средний срок - 12 лет)
  - Современная инфраструктура (30,500 кв.м складов)
- Иконки Font Awesome для визуального усиления
- Hover-эффекты с плавной анимацией

#### 3. **Ключевые показатели**
- 6 анимированных счетчиков с данными:
  - 30 лет опыта
  - 500 сотрудников
  - 150,000 заказов в год
  - 30,500 кв.м складов
  - 12 лет среднее партнерство
  - 98% охват рынка Жезказгана
- Автоматический подсчет при прокрутке (Intersection Observer)
- Темный фон с градиентом для контраста

#### 4. **География присутствия**
- 4 города: Караганда, Жезказган (featured), Сатпаев, Астана
- Детальная статистика по ключевым рынкам
- Выделенная карточка для Жезказгана (98% охват)
- Призыв к масштабированию бизнеса

#### 5. **Компетенции и услуги**
- 4 основных направления с изображениями из поиска:
  - **Дистрибуция и логистика** - современные складские комплексы
  - **Оптовая торговля FMCG** - крупные объемы обработки
  - **Температурные режимы хранения** - холодильное оборудование
  - **Профессиональная команда** - опытные специалисты

#### 6. **Вертикальная интеграция**

  5. Розница (супермаркеты + гипермаркет 4,000 кв.м)

#### 7. **Партнеры**

#### 8. **Контактная форма**
- Двухколоночный layout (информация + форма)
- 4 поля: имя, email, телефон, сообщение
- Валидация в реальном времени

## 🚀 Запуск проекта

### Локальный запуск

1. **Клонирование/скачивание файлов**
   ```bash
   # Если используется Git
   git clone <repository-url>
   cd samadi-landing
   ```

2. **Открытие в браузере**
   
   ⚠️ **ВАЖНО**: Для корректного отображения стилей и изображений **необходимо использовать локальный веб-сервер**!
   
   **Простое открытие файла (file://) может не работать из-за ограничений CORS.**
   
   ### Рекомендуемые способы запуска:
   
   **Способ 1: Python (самый простой)**
   ```bash
   # Откройте терминал в папке проекта
   python -m http.server 8000
   # Или для Python 2
   python -m SimpleHTTPServer 8000
   ```
   Затем откройте: `http://localhost:8000`
   
   **Способ 2: Node.js**
   ```bash
   # Установите http-server глобально (один раз)
   npm install -g http-server
   
   # Запустите сервер
   http-server
   ```
   Затем откройте: `http://localhost:8080`
   
   **Способ 3: VS Code (рекомендуется для разработки)**
   1. Установите расширение "Live Server" в VS Code
   2. Откройте папку проекта в VS Code
   3. Правый клик на `index.html` → "Open with Live Server"
   4. Автоматически откроется браузер
   
   **Способ 4: PHP**
   ```bash
   php -S localhost:8000
   ```

3. **Доступ к сайту**
   - После запуска сервера откройте указанный URL в браузере
   - Все стили, скрипты и изображения загрузятся корректно

### Деплой на хостинг

#### Опция 1: Netlify (рекомендуется)
1. Зарегистрируйтесь на [Netlify](https://www.netlify.com/)
2. Перетащите папку проекта в Netlify Drop
3. Готово! Получите автоматический URL

#### Опция 2: Vercel
1. Зарегистрируйтесь на [Vercel](https://vercel.com/)
2. Импортируйте проект из Git или загрузите файлы
3. Deploy готов за минуту

#### Опция 3: GitHub Pages
1. Создайте репозиторий на GitHub
2. Загрузите файлы
3. Settings → Pages → Deploy from branch (main)
