## ⚠️ ВАЖНО!

**Сайт НЕ будет работать при простом открытии файла `index.html` двойным кликом!**

Вы увидите только текст без стилей и изображений (как на вашем скриншоте).

**Причина**: Современные браузеры блокируют загрузку локальных ресурсов (CSS, JavaScript, изображения) по протоколу `file://` из-за политики безопасности CORS.

## ✅ ПРАВИЛЬНЫЙ СПОСОБ ЗАПУСКА

### Вариант 1: Python (САМЫЙ ПРОСТОЙ) 🐍

**Для Windows, Mac, Linux:**

1. Откройте **Терминал** (или **Командную строку** на Windows)
2. Перейдите в папку с проектом:
   ```
   cd путь/к/папке/samadi-landing
   ```
3. Запустите команду:
   ```
   python -m http.server 8000
   ```
   Или для Python 2:
   ```
   python -m SimpleHTTPServer 8000
   ```

4. Откройте браузер и перейдите по адресу:
   ```
   http://localhost:8000
   ```

5. **Готово!** Сайт откроется со всеми стилями и функциями.

Для остановки сервера: нажмите `Ctrl+C` в терминале.

---

### Вариант 2: VS Code + Live Server (ДЛЯ РАЗРАБОТЧИКОВ) 💻

**Если у вас установлен VS Code:**

1. Откройте **VS Code**
2. Установите расширение "**Live Server**" (автор: Ritwick Dey)
3. Откройте папку проекта: `File` → `Open Folder` → выберите папку `samadi-landing`
4. Найдите файл `index.html` в боковой панели
5. **Правый клик** на `index.html` → `Open with Live Server`
6. Браузер автоматически откроется с сайтом
7. **Бонус**: При изменении файлов страница автоматически обновляется!

---

### Вариант 3: Node.js + http-server 🟢

**Если у вас установлен Node.js:**

1. Откройте терминал в папке проекта
2. Установите http-server (один раз):
   ```
   npm install -g http-server
   ```
3. Запустите сервер:
   ```
   http-server
   ```
4. Откройте браузер:
   ```
   http://localhost:8080
   ```

---

### Вариант 4: PHP 🐘

**Если у вас установлен PHP:**

1. Откройте терминал в папке проекта
2. Запустите:
   ```
   php -S localhost:8000
   ```
3. Откройте браузер:
   ```
   http://localhost:8000
   ```

## 🌐 ДЕПЛОЙ НА РЕАЛЬНЫЙ ХОСТИНГ

### Netlify (Рекомендуется - БЕСПЛАТНО)

1. Зайдите на [netlify.com](https://www.netlify.com/)
2. Зарегистрируйтесь (можно через GitHub)
3. Перетащите **всю папку** проекта в область "**Drag and drop your site folder here**"
4. Подождите 10-20 секунд
5. **Готово!** Получите ссылку типа `https://samadi-kz.netlify.app`

**Преимущества:**
- ✅ Бесплатный SSL сертификат
- ✅ Автоматические обновления
- ✅ CDN для быстрой загрузки
- ✅ Можно подключить свой домен

### Vercel (Альтернатива)

1. Зайдите на [vercel.com](https://vercel.com/)
2. Зарегистрируйтесь
3. Загрузите папку проекта
4. Автоматический деплой
5. Получите ссылку типа `https://samadi.vercel.app`

### GitHub Pages (Если используете Git)

1. Создайте репозиторий на GitHub
2. Загрузите файлы проекта
3. Settings → Pages → Deploy from branch (main)
4. Ссылка: `https://ваш-username.github.io/samadi-landing`

## 📞 ПОМОЩЬ

Если ничего не работает:

1. **Проверьте Python**: в терминале введите `python --version`
2. **Попробуйте другой браузер** (Chrome, Firefox, Safari)
3. **Очистите кэш браузера**: `Ctrl+Shift+R` (Windows) или `Cmd+Shift+R` (Mac)
4. **Проверьте файлы**: убедитесь, что папка `css` и `js` находятся в той же папке, что и `index.html`
