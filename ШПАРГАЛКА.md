## 🔗 ИНТЕГРАЦИЯ С GITHUB (10 минут)

### 1. Создайте репозиторий на GitHub

```
github.com → New repository → samadi-landing → Create
```

### 2. Загрузите файлы

**Через веб:**
```
Add file → Upload files → Перетащите все → Commit
```

**Через Git:**
```bash
git init
git add .
git commit -m "Initial commit"
git remote add origin https://github.com/USERNAME/samadi-landing.git
git push -u origin main
```

### 3. Подключите Netlify

```
app.netlify.com → Add new site → Import from GitHub
→ Выберите репозиторий → Deploy site
```

**Результат:** Автодеплой при каждом изменении! 🎯

## 🆘 ПРОБЛЕМЫ

### "Стили не загружаются"
→ Используйте Netlify вместо прямого открытия файла

### "GitHub спрашивает пароль"
→ Создайте Personal Access Token

### "Netlify показывает 404"
→ Проверьте, что index.html в корне

### "Долго грузится"
→ Очистите кэш: Ctrl+Shift+R
